# Shimmer Granular

Гранулярный шиммер-ревербератор в форматах VST3 и CLAP, написанный на Rust с использованием фреймворка [nih-plug](https://github.com/robbert-vanhelmholdt/nih-plug).

---

## Схема блоков

### Уровень 1 — Архитектура

```
                          ┌──────────────────────┐
                          │  РАННИЕ ОТРАЖЕНИЯ    │
           ВХОД (L/R) ───►│  (8 sparse taps)     ├─────────────────────► early
                    │     └──────────────────────┘
                    │
                    │     ┌──────────────────────┐
                    └────►│                      │
                    ▲     │    РЕВЕРБЕРАЦИЯ      ├─────────────────────► tail
                    │     │   (FDN 8×8 Hadamard) │
                    │     └──────────────────────┘
                    │               │ tail
                    │     ┌─────────▼────────────┐
                    │     │   ПИТЧ-ШИФТЕР        ├─────────────────────► grain
                    │     │   (гранулятор)       │
                    │     └──────────────────────┘
                    │
                    └──────────── Regen × grain
                        (shimmer loop: каждый проход
                         поднимает тон на pitch_shift)

  early ────────────────────────────────────────┐
  tail  ────────────────────────────────────────┤
  grain ────────────────────────────────────────┼──► WET MIX ──► DRY/WET ──► GAIN ──► ВЫХОД
  ВХОД (dry) ───────────────────────────────────┘
```

### Уровень 2 — Детальная схема

```
  ВХОД (L/R) ───────────────────────────────────────────────────────────────────┐
       │                                                                         │
       │   ╔══════════════════════════════════════════════════╗                  │
       │   ║  РАННИЕ ОТРАЖЕНИЯ (EarlyReflections)             ║                  │
       │   ║                                                  ║                  │
       ├──►║  8 sparse taps ~5..77 мс                         ║──► early         │
       │   ║  убывающие gain 0.70..0.25                       ║                  │
       │   ╚══════════════════════════════════════════════════╝                  │
       │                                                                         │
       ▼                                                                         │
  ┌────────────────────┐                                                         │
  │  СУММИРОВАНИЕ      │◄────── Regen × grain  (soft_clip · DC-блок)             │
  └────────┬───────────┘                                                         │
           │                                                                     │
           │  ╔══════════════════════════════════════════════════════╗            │
           │  ║  РЕВЕРБЕРАЦИЯ  (FDN 8×8)                             ║            │
           │  ║                                                      ║            │
           └─►║  Allpass-декоррелятор R-канала (~0.5 мс)            ║            │
              ║         ▼                                            ║            │
              ║  8 линий задержки (взаимно простые ~15..43 мс)      ║            │
              ║         ▼                                            ║            │
              ║  Fast Hadamard Transform 8×8  (только +/−)          ║            │
              ║         ▼                                            ║            │
              ║  Absorption: OnePoleLP × 8  (Diffusion → темнее)    ║            │
              ║         ▼                                            ║            │
              ║  Feedback gain × 8  (Tail Feedback)                 ║            │
              ║         ▼                                            ║            │
              ║  HPF (20–500 Гц)   LPF (1000–20000 Гц)              ║            │
              ║                                                      ║            │
              ║  L = линии [0,2,4,6]   R = линии [1,3,5,7]          ║            │
              ╚══════════════════════════╦═══════════════════════════╝            │
                                        │ tail L/R                               │
              ╔═════════════════════════╩═══════════════════════════╗            │
              ║  ПИТЧ-ШИФТЕР (гранулятор)                           ║            │
              ║                                                      ║            │
              ║  Кольцевой буфер  2 с · Catmull-Rom                 ║            │
              ║         ▼                                            ║            │
              ║  Планировщик  Density · Pos Jitter                   ║            │
              ║         ▼                                            ║            │
              ║  Скорость  2^(st/12 + ¢/1200)                       ║            │
              ║         ▼                                            ║            │
              ║  Огибающая  AAHT-R · кривые Atk/Rel                 ║            │
              ║         ▼                                            ║            │
              ║  Нормализация  1/N зёрен                            ║            │
              ╚══════════════════════════╦══════════════════════════╝            │
                                        │ grain                                  │
                                        ├──── × Regen ──────────────────────────┘
                                        │
                         ┌──────────────▼──────────────────────┐
                         │  WET MIX                             │
                         │  = 0.3·early + 0.35·tail + 0.35·grain│
                         └──────────────┬──────────────────────┘
                                        │              ┌─── ВХОД (dry)
                                        ▼              ▼
                         ┌─────────────────────────────────────┐
                         │  DRY/WET   equal-power cos/sin · π/2 │
                         └──────────────┬──────────────────────┘
                                        ▼
                         ┌──────────────────────────┐
                         │  OUTPUT GAIN  −24..+12 дБ │
                         └──────────────┬────────────┘
                                        ▼
                                    ВЫХОД (L/R)
```

> **Shimmer loop:** гранулярный выход умножается на **Regen** и подаётся обратно на вход реверберации. Каждый проход через питч-шифтер поднимает сигнал на `Pitch Shift` полутонов — так накапливаются гармоники и рождается характерный «шиммер».
>
> **Стерео-декорреляция:** FDN 8×8 разделяет L (линии 0,2,4,6) и R (линии 1,3,5,7). Матрица Адамара перемешивает их при каждом проходе — стерео-ширина без явной задержки. Дополнительный allpass на R-канале перед входом в FDN разрушает монофоническую симметрию входного сигнала.

## Параметры

### GRAIN — Гранулятор

| Параметр | Диапазон | По умолчанию | Ед. | Описание |
|---|---|---|---|---|
| **Grain Size** | 10 – 400 мс | 60 мс | мс | Длина одного зерна. Малые значения дают «зернистую» текстуру, большие — плавный размытый звук. |
| **Pitch Shift** | −36 … +36 пт | +12 пт | пт | Транспонирование зёрен. +12 даёт классический шиммер на октаву вверх. |
| **Density** | 10 – 100 з/с | 20 з/с | з/с | Количество зёрен в секунду. Чем выше плотность, тем ровнее звук. |
| **Pos Jitter** | 0 – 100 % | 20 % | % | Случайное смещение позиции чтения зерна. Увеличивает диффузию и убирает периодические артефакты. |

### TAIL — Хвост реверберации

| Параметр | Диапазон | По умолчанию | Ед. | Описание |
|---|---|---|---|---|
| **Tail Time** | 50 – 2000 мс | 480 мс | мс | Время затухания реверберации. |
| **Tail Feedback** | 0 – 95 % | 55 % | % | Обратная связь внутри хвоста. Прямо влияет на длину «хвоста». |
| **Diffusion** | 0 – 100 % | 60 % | % | Тональное поглощение LP-фильтров внутри FDN. Высокие значения делают хвост темнее и размывают атаки. |
| **Modulation** | 0 – 10 мс | 2 мс | мс | Глубина LFO-модуляции хвоста (синус 0.37 Гц). Добавляет хорус-эффект и подавляет металлические резонансы. |

### TONE — Тональный баланс

| Параметр | Диапазон | По умолчанию | Ед. | Описание |
|---|---|---|---|---|
| **Tail HPF** | 20 – 500 Гц | 120 Гц | Гц | Однополюсный фильтр высоких частот на хвосте реверберации. Убирает «мутность» в НЧ. |
| **Tail LPF** | 1000 – 20000 Гц | 6000 Гц | Гц | Однополюсный фильтр низких частот на хвосте реверберации. Смягчает яркость. |
| **Pitch Drift** | 0 – 400 центов | 0 центов | центов | Случайная расстройка каждого зерна в пределах ±значение. Добавляет «живость» и хорус без изменения общей высоты. |
| **Regen** | 0 – 95 % | 35 % | % | Уровень, с которым питч-сдвинутый гранулярный выход подаётся обратно на вход хвоста. Главная «ручка шиммера». |

### ENVELOPE — Огибающая зерна

Огибающая имеет форму AAHT-R: атака → плато → держание → спад.

| Параметр | Диапазон | По умолчанию | Ед. | Описание |
|---|---|---|---|---|
| **Attack** | 0 – 100 % | 50 % | % | Доля длины зерна, занятая фазой нарастания. |
| **Hold** | 0 – 100 % | 0 % | % | Доля длины зерна в фазе плато (атака + держание ≤ 100 %). |
| **Attack Curve** | −1 … +1 | 0 | — | Форма нарастания: −1 = логарифм (быстрый подъём), 0 = линейная, +1 = экспонента (медленный подъём). |
| **Release Curve** | −1 … +1 | +1 | — | Форма спада: −1 = логарифм (быстрое затухание), 0 = линейная, +1 = экспонента (плавное затухание). |

**Пресеты огибающей:**

| Пресет | Attack | Hold | A Curve | R Curve | Применение |
|---|---|---|---|---|---|
| Hann | 50 % | 0 % | Linear | Linear | Классическое гранулярное окно, минимальные артефакты |
| Tukey | 15 % | 70 % | Linear | Linear | Плотная плита звука с короткими скосами |
| Smooth | 40 % | 10 % | −0.5 | +0.5 | Мягкий вход, медленное затухание |
| Perc | 1 % | 0 % | Linear | +0.7 | Острая атака с длинным шлейфом, для ударных текстур |

### OUTPUT — Выходной уровень

| Параметр | Диапазон | По умолчанию | Ед. | Описание |
|---|---|---|---|---|
| **Dry/Wet** | 0 – 100 % | 50 % | % | Баланс сухого и обработанного сигнала (equal-power crossfade). |
| **Gain** | −24 … +12 дБ | 0 дБ | дБ | Выходное усиление. |

## Быстрые пресеты высоты тона

Кнопки на панели позволяют мгновенно задать Pitch Shift:

| Кнопка | Значение | Эффект |
|---|---|---|
| −12 | −12 пт | Шиммер на октаву вниз |
| −6 | −6 пт | Квинта вниз |
| 0 | 0 пт | Унисон (только ревербератор без питч-сдвига) |
| +6 | +6 пт | Квинта вверх |
| +12 | +12 пт | Стандартный шиммер (октава вверх) |

## Сборка

Требуется Rust (edition 2024, стабильный тулчейн).

```bash
# VST3 и CLAP в папку target/bundled/
cargo xtask bundle shimmer-vst --release
```

Готовые бинари появятся в `target/bundled/`.
